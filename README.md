# NestGPT Api

## Backend de la app ReactGPT

## Tecnologias

- Nest 10.0.0
- Node 20.9.0

## Getting Started

### Generate the .env file based on .env.template file.

### Put your `OPENAI_API_KEY` <https://platform.openai.com/api-keys>

### Install dependencies:

```bash
nvm use 20.9.0

pnpm install
```

### Start application in development mode:

```bash
nvm use 20.9.0

pnpm run start:dev
```
